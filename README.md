# Inventaris prog om ... een inventaris bij te houden (via xlsx).
Eem web/php toepassing om een inventaris via xlsx bij te houden.
De spreadsheet, met overeenkomstige JSON en CSV versies wordt op
de webserver centraal bijgehouden. Er is een mogelijkheid om de
gegevens te downloaden.
Er komt ook een versiebeheer, zodat de oude toestand bijgehouden
wordt als een nieuwe versie bewaard wordt.
